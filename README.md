# Ejercicios practicos — apps Flask separadas

Dos aplicaciones independientes (cada una con su venv y su base de datos):

- `gestor_tu_espacio/` — Tu espacio, calendario, Trading Lab (puerto tipico 5000).
- `gestor_historia/` — Simulador Historia / nucleo (puerto tipico 5001).

El monolito `gestor_tareas` fue retirado. Si queda una carpeta `gestor_tareas` con venv bloqueado, cierra Python y borrala o ejecuta `eliminar_gestor_tareas_restante.ps1`.
